# 🚀 Credit Risk Analysis & Prediction
## 📌 Overview
This project focuses on analyzing financial data to predict the probability of loan default using Machine Learning techniques.
## 🎯 Objective
To identify key factors influencing credit risk and build a predictive model for classification.
## 🛠️ Tech Stack
* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn
## 🔍 Workflow
* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Model Building (Logistic Regression)
* Model Evaluation
## 📊 Results
* Achieved **93% accuracy**
* Strong performance across precision, recall, and F1-score
## 💡 Key Insights
* Lower income → Higher default risk
* Higher loan amount → Increased probability of default
* Data-driven models improve financial decision-making
## 📁 Files Included
* Jupyter Notebook with full analysis
* Dataset (optional)
## 🚀 How to Run
```bash
pip install -r requirements.txt
jupyter notebook
```
